Formatting

HTML contains several elements for defining text with a special meaning. Below is a list of formatting elements:

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

This is a blockquote tag, <blockquote>. It is for large quotes.

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

This is a quote tag, <q>. It is for short quotes. It also applies double quotes.

WWF's goal is to: Build a future where people live in harmony with nature.

This is an abbreviation tag, <abbr>. It makes it easier for translations and search engines.

The WHO was founded in 1948.

This is an address tag, <address>.

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

This is a citation tag, <cite>.

The Scream by Edvard Munch. Painted in 1893.

This is a Bi-Directional Override Tag, <bdo>. It flips the words around.

This text will be written from right to left

This text will be written from right to left
