Tables

Here is an example of the table tag:

Name Book/Series Book Genre
Harry Potter Harry Potter Fantasy/Magic
Frodo Baggins Lord of the Rings Fantasy/Magic
Percy Jackson Percy Jackson Fantasy/Mythology
Sherlock Holmes Sherlock Holmes Mystery

Tables can be created by doing this:

<table>
<tr>
<th>Category One</tr>

If you want more categories, repeat the same thing replacing Item One with something else.

<table>
<tr>
<th>Category Two</tr>

When you are satisfied with the amount of categories, you can start placing things into these cateories

<tr>
<td>Item One </td>
<td>Item Two</td>
</tr>
<tr>
<td>Item One.1 </td>
<td>Item Two.1</td>
</tr>
<tr>
<td>Item One.2 </td>
<td>Item Two.2</td>
</tr>

The end product should look like this:

Category One Category Two
Item One Item Two
Item One.1 Item Two.1
Item One.2 Item Two.2
